# Vision

Aetheris is being built as a modern solid modeling kernel first, with clear boundaries between geometric computation and higher-level product features.

## Direction

- Authoritative representation: B-rep is the source of truth for modeled solids.
- Kernel scope: robust geometric/topological operations and diagnostics.
- Parametric/history: intentionally outside the kernel boundary.
- Long-term foundation: a continuum-unified substrate is a strategic goal, not an early milestone deliverable.
- Execution target: server/web-first deployment and integration.
- Persistence/interchange: AP242 is the primary target, mapped at boundaries rather than mirrored as in-memory domain objects.
